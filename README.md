# cloud

A web-based air quality monitoring station firmware.

## Requirements ##

- **NodeMCU** board [ station core ]
- **BME280** sensor [ temperature, humidity, pressure ]
- **MQ-9** sensor [ CO concentration ]
- **PMS7003** sensor [ PM1, PM2.5, PM10 dust concentration ]
